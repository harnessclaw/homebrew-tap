# Homebrew Tap for HarnessClaw

Official Homebrew tap for [HarnessClaw](https://github.com/harnessclaw).

## Installation

```bash
brew tap harnessclaw/tap
brew install --cask harnessclaw
```

## Update

```bash
brew update
brew upgrade --cask harnessclaw
```

## Uninstall

```bash
brew uninstall --cask harnessclaw
brew untap harnessclaw/tap
```

## Troubleshooting

If you encounter issues during installation:

```bash
# Force reinstall
brew reinstall --cask harnessclaw

# Clear cache and reinstall
brew cleanup
brew install --cask harnessclaw
```
